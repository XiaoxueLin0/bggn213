\subsection{to work with any dataset of protein
structure..}\label{to-work-with-any-dataset-of-protein-structure..}

The input of the function reads protein pdb files. The Output is the B
factor plot of protein. The following function first takes pdb of a
protein with \textbf{read.pdb}, then uses \textbf{trim.pdb} to focus on
chain A and ``CA''" atom type. \textbf{plotb3} will plot the
protein-drug interaction of selected sequence with B factor, and gives
the output of one or more than one plots.

\begin{Shaded}
\begin{Highlighting}[]
\KeywordTok{library}\NormalTok{(bio3d)}
\CommentTok{# name "plot"" as the function}
\NormalTok{plot <-}\StringTok{ }\ControlFlowTok{function}\NormalTok{ (x)  \{}
  \ControlFlowTok{for}\NormalTok{ (protein }\ControlFlowTok{in}\NormalTok{ x) \{}
  \CommentTok{# s call s as read.pdb}
\NormalTok{  s <-}\StringTok{ }\KeywordTok{read.pdb}\NormalTok{(protein)}
  \CommentTok{# trim.pdb produces a new smaller PDB object (protein chains) with a subset of atoms from a given larger PDB object.}
  \CommentTok{# call s.chainA to use trim.pdb function to select chain A }
\NormalTok{  s.chainA <-}\StringTok{ }\KeywordTok{trim.pdb}\NormalTok{(s, }\DataTypeTok{chain=}\StringTok{"A"}\NormalTok{, }\DataTypeTok{elety=}\StringTok{"CA"}\NormalTok{)}
  \CommentTok{# s.b places amino acid in **B factor**}
\NormalTok{  s.b <-}\StringTok{ }\NormalTok{s.chainA}\OperatorTok{$}\NormalTok{atom}\OperatorTok{$}\NormalTok{b}
  \CommentTok{# plot the data with amino acid on x axis and B factor (amino acid mobility) on y axis}
  \KeywordTok{plotb3}\NormalTok{(s.b, }\DataTypeTok{sse=}\NormalTok{s.chainA, }\DataTypeTok{type =} \StringTok{"l"}\NormalTok{, }\DataTypeTok{ylab =} \StringTok{"Bfactor"}\NormalTok{)}
\NormalTok{  \}}
\NormalTok{\}}
\KeywordTok{plot}\NormalTok{(}\KeywordTok{c}\NormalTok{(}\StringTok{"4AKE"}\NormalTok{, }\StringTok{"1AKE"}\NormalTok{, }\StringTok{"1E4Y"}\NormalTok{))}
\end{Highlighting}
